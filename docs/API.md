# API

Base URL: `http://localhost:5000/api` during local development.

Public endpoints include health, fish catalogue, customer enquiries, chat sessions, care guides, and site settings. Protected endpoints include fish mutations, media uploads, order management, dashboard metrics, care-guide management, settings updates, and admin chat operations.

Admin authentication uses the HTTP-only `sj_admin_token` cookie. Use the ready-made Postman collection in the `postman` folder for request bodies and examples.
